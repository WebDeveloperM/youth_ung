# ✅ КООРДИНАТОР МОЖЕТ РЕДАКТИРОВАТЬ ПОЛЬЗОВАТЕЛЕЙ!

## 🎯 ЧТО РЕАЛИЗОВАНО:

### 1. ✅ Координатор может редактировать пользователей своей организации

Координатор теперь может изменять следующие данные пользователей:

- **Основная информация:**
  - ✅ Имя (first_name)
  - ✅ Фамилия (last_name)
  - ✅ Телефон (phone)
  - ✅ Дата рождения (date_of_birth)
  - ✅ Пол (gender)
  
- **Дополнительная информация:**
  - ✅ Должность (position)
  - ✅ Адрес (address)
  - ✅ Аватар (avatar)

### 2. ❌ Координатор НЕ МОЖЕТ изменять:

- ❌ Логин (username)
- ❌ Email
- ❌ Пароль (password)
- ❌ Роль (role)
- ❌ Организацию (organization)
- ❌ Статус активности (is_active)

### 3. ✅ Ограничения доступа:

- ✅ Координатор видит ТОЛЬКО обычных пользователей (role=User) своей организации
- ✅ НЕ видит админов, модераторов и других координаторов
- ✅ НЕ видит пользователей других организаций
- ✅ Может редактировать ТОЛЬКО тех кого видит

## 📋 API ENDPOINTS:

### GET /api/v1/admin/all-users/
**Получить список пользователей**

**Для координатора:**
```json
{
  "count": 3,
  "results": [
    {
      "id": 3,
      "username": "davlat_user",
      "email": "user@mgpz.uz",
      "role": "User",  // ← Только User!
      "organization_name": "MGPZ",  // ← Только своя организация!
      "position": "Специалист",
      "date_of_birth": "1995-05-15",
      // ...
    }
  ]
}
```

### PATCH /api/v1/admin/all-users/{id}/
**Обновить пользователя**

**Request:**
```json
{
  "first_name": "Давлат Updated",
  "position": "Старший специалист",
  "address": "Ташкент, ул. Навои 123"
}
```

**Response (200 OK):**
```json
{
  "id": 3,
  "first_name": "Давлат Updated",
  "last_name": "Раупов",
  "position": "Старший специалист",
  "address": "Ташкент, ул. Навои 123",
  // ... остальные поля ...
}
```

**Error (403 Forbidden):**
```json
{
  "error": "Вы можете редактировать только пользователей своей организации"
}
```

или

```json
{
  "error": "Вы можете редактировать только обычных пользователей"
}
```
