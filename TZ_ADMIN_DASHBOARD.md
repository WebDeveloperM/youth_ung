# ТЕХНИЧЕСКОЕ ЗАДАНИЕ
## Административная панель (Dashboard) для UNG Youth Platform

---

## 📋 ОБЩАЯ ИНФОРМАЦИЯ

**Проект:** UNG Youth Platform - платформа для молодежи Узбекистана  
**Цель:** Создание полнофункциональной административной панели для управления всем контентом и пользователями платформы  
**Технологии:**
- **Frontend (Admin Dashboard):** React + TypeScript + Tailwind CSS + Recharts
- **Backend:** Django REST Framework
- **База данных:** PostgreSQL

---

## 🎯 ОСНОВНЫЕ ТРЕБОВАНИЯ

Админ панель должна обеспечивать **ПОЛНОЕ управление** всем контентом платформы, включая:
- Управление публикациями (8 типов контента)
- Управление пользователями и ролями
- Управление комментариями и модерация
- Аналитика и статистика
- Управление организациями
- Настройки платформы

---

### 3.1 Список пользователей

**Таблица:**

**Массовые действия:**
- Изменить роль
- Активировать/Деактивировать
- Удалить

### 3.2 Создание/Редактирование пользователя

**Личная информация:**
- Имя - обязательно
- Фамилия - обязательно
- Email - обязательно, уникальный
- Телефон - обязательно
- Дата рождения
- Пол
- Адрес
- Аватар (загрузка)

**Организационная информация:**
- Организация (выбор из списка) - обязательно
- Должность

**Учетная запись:**
- Username (автогенерация или ручной ввод)
- Пароль (при создании - обязательно, при редактировании - опционально для смены)
- Роль (Admin/Moderator/User) - обязательно
- Статус (активный/неактивный)

**Статистика и активность:**
- Дата регистрации
- Последний вход
- Количество комментариев
- Количество лайков
- История активности (просмотр)

**Действия:**
- Сохранить
- Сохранить и создать еще
- Отправить письмо для сброса пароля
- Войти как этот пользователь (для тестирования)
- Деактивировать/Активировать
- Удалить (с подтверждением)

### 3.3 Роли и права доступа

**Роли:**
1. **User (Пользователь):**
   - Просмотр контента
   - Добавление комментариев
   - Лайки
   - Редактирование своего профиля

2. **Moderator (Модератор):**
   - Все права User +
   - Модерация комментариев (одобрение/удаление)
   - Просмотр статистики
   - Ответы на комментарии от имени администрации

3. **Admin (Администратор):**
   - Все права Moderator +
   - Создание/редактирование всего контента
   - Управление пользователями
   - Управление организациями
   - Доступ к полной аналитике
   - Настройки системы

**Настройка прав (опционально для будущего):**
- Детальная настройка прав для каждой роли
- Создание кастомных ролей

## 📊 5. АНАЛИТИКА

### 5.1 Общая статистика

**Dashboard аналитики:**
- **Период:** Выбор диапазона дат или предустановки (сегодня, неделя, месяц, год, все время)

**Метрики:**
1. **Посетители:**
   - Всего посетителей
   - Уникальные посетители
   - Новые посетители
   - Возвращающиеся посетители
   - График посетителей по дням
   - Распределение по часам (heatmap)

2. **Просмотры:**
   - Всего просмотров страниц
   - Среднее страниц на посетителя
   - Среднее время на сайте
   - Показатель отказов (bounce rate)

3. **Устройства:**
   - Мобильные (%)
   - Планшеты (%)
   - Десктоп (%)
   - Pie Chart распределения

4. **География:**
   - Топ стран
   - Топ городов
   - Карта посетителей (если есть данные)

5. **Источники трафика:**
   - Прямые переходы
   - Поисковые системы
   - Социальные сети
   - Рефералы (другие сайты)

### 5.2 Аналитика контента

**Топ контента:**
- **Самые просматриваемые:**
  - Новости (топ 10)
  - Гранты (топ 10)
  - Инновации (топ 10)
  - И т.д. для всех типов
  - Метрики: просмотры, уникальные посетители, среднее время, лайки

- **Самые комментируемые:**
  - Список публикаций с наибольшим количеством комментариев
  - Метрики: количество комментариев, средний рейтинг

- **Trending (в тренде):**
  - Контент с наибольшим ростом просмотров за последние 7 дней

**Детальная аналитика публикации:**
- График просмотров по дням
- Источники трафика для этой публикации
- География посетителей
- Устройства
- Время на странице
- Количество комментариев по дням
- Динамика лайков

### 5.3 Аналитика пользователей

**Метрики:**
- Регистрации по дням (график)
- Активные пользователи (DAU, WAU, MAU)
- Топ активных пользователей (по комментариям, лайкам)
- Retention rate (возвращаемость пользователей)
- Churn rate (отток пользователей)

**Сегментация:**

**Endpoints для админки:**

**Контент:**
- `GET /api/v1/admin/news/` - список новостей
- `POST /api/v1/admin/news/` - создать новость
- `GET /api/v1/admin/news/{id}/` - получить новость
- `PUT/PATCH /api/v1/admin/news/{id}/` - обновить новость
- `DELETE /api/v1/admin/news/{id}/` - удалить новость
- `POST /api/v1/admin/news/bulk-action/` - массовые действия

Аналогично для: grants, scholarships, competitions, innovations, internships, jobs, team

**Пользователи:**
- `GET /api/v1/admin/users/` - список пользователей
- `POST /api/v1/admin/users/` - создать пользователя
- `GET /api/v1/admin/users/{id}/` - получить пользователя
- `PUT/PATCH /api/v1/admin/users/{id}/` - обновить пользователя
- `DELETE /api/v1/admin/users/{id}/` - удалить пользователя
- `POST /api/v1/admin/users/{id}/change-role/` - изменить роль
- `POST /api/v1/admin/users/{id}/reset-password/` - сбросить пароль
- `GET /api/v1/admin/users/{id}/activity/` - история активности

**Комментарии:**
- `GET /api/v1/admin/comments/` - список всех комментариев
- `GET /api/v1/admin/comments/{id}/` - получить комментарий
- `PATCH /api/v1/admin/comments/{id}/` - модерировать
- `DELETE /api/v1/admin/comments/{id}/` - удалить
- `POST /api/v1/admin/comments/bulk-moderate/` - массовая модерация

**Аналитика:**
- `GET /api/v1/admin/analytics/overview/` - общая статистика
- `GET /api/v1/admin/analytics/visitors/` - статистика посетителей
- `GET /api/v1/admin/analytics/content/` - статистика контента
- `GET /api/v1/admin/analytics/users/` - статистика пользователей
- `GET /api/v1/admin/analytics/export/` - экспорт данных

**Организации:**
- CRUD endpoints для organisations, departments, sections

**Настройки:**
- `GET /api/v1/admin/settings/` - получить настройки
- `PATCH /api/v1/admin/settings/` - обновить настройки
