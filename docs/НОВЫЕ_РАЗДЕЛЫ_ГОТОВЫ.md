## 📍 API ENDPOINTS

### Публичное API (для frontend):

**Technologies:**
- `GET /api/v1/technologies/` - список технологий
- `GET /api/v1/technologies/<id>/` - детальная технология
- `POST /api/v1/technologies/<id>/like/` - лайкнуть

**Projects:**
- `GET /api/v1/projects/` - список проектов
- `GET /api/v1/projects/<id>/` - детальный проект
- `POST /api/v1/projects/<id>/like/` - лайкнуть

**Research:**
- `GET /api/v1/research/` - список исследований
- `GET /api/v1/research/<id>/` - детальное исследование
- `POST /api/v1/research/<id>/like/` - лайкнуть

**Results:**
- `GET /api/v1/results/` - список результатов
- `GET /api/v1/results/<id>/` - детальный результат
- `POST /api/v1/results/<id>/like/` - лайкнуть

### Фильтры:
- `?category=<category>` - по категории
- `?status=<status>` - по статусу  
- `?year=<year>` - по году (только для results)
- `?is_featured=true` - избранные
- `?limit=<n>` - ограничение количества
