## Как использовать

### В Django Admin
1. Перейдите в раздел "Команда"
2. При создании/редактировании члена команды выберите категорию из выпадающего списка
3. Категория отображается в списке и доступна для фильтрации

### В Админ-панели (Dashboard)
1. Откройте раздел "Жамоа" (Команда)
2. При добавлении/редактировании члена команды выберите категорию
3. Категория отображается в таблице со специальным бейджем

### На публичном сайте
1. Посетители могут фильтровать членов команды по категориям
2. Кнопки фильтрации находятся под заголовком страницы
3. Можно выбрать "Все" или конкретную категорию

## Миграция базы данных

Для применения изменений в базе данных выполните:

```bash
cd backend
python manage.py migrate content
```

Миграция автоматически установит значение по умолчанию `leadership` для всех существующих членов команды.
