## 🚀 КАК ИСПОЛЬЗОВАТЬ

### DASHBOARD (Админка):

#### Создание гранта:
1. Открой `http://localhost:5173` (dashboard)
2. Войди: `admin@admin.com` / `admin123`
3. Перейди на **"Грантлар"**
4. Нажми **"Yangi grant"**
5. Заполни форму:
   - **Расм** (опционально)
   - **Сумма** (например: "10,000 USD")
   - **Давомийлик** (например: "6 месяцев")
   - **Дедлайн** (выбери дату)
   - **Статус** (Активный/Скоро/Закрыт)
   - **Категория** (Инновации/Экология/Цифровые/Социальные)
6. **Переключайся между UZ/RU/EN** и заполняй:
   - Номи (название)
   - Қисқача тавсиф (краткое описание)
   - Тўлиқ тавсиф (полное описание в Rich Text Editor)
7. Нажми **"Қўшиш"**
8. **Проверь консоль (F12)** - должны быть логи с данными!

#### Редактирование гранта:
1. Нажми ✏️ рядом с грантом
2. **Подожди 1-2 секунды** - загружается полная модель
3. **Проверь консоль** - должно быть "ЗАГРУЖЕН ПОЛНЫЙ ГРАНТ"
4. Переключись между вкладками - **тексты видны!**
5. Редактируй и нажми "Сақлаш"

### FRONTEND (Основной сайт):

#### Просмотр грантов:
1. Открой `http://localhost:5174` (frontend)
2. Перейди на **"/grants"**
3. Увидишь **все гранты из базы данных!**
4. Можешь:
   - Сортировать по статусу/категории
   - Видеть amount, duration, deadline
   - Переходить на детальную страницу

#### Детальная страница гранта:
1. Кликни на грант
2. Увидишь:
   - Полное описание на выбранном языке
   - Изображение (или иконку 💰)
   - Все параметры гранта
   - Кнопку "Подать заявку" (если активный)

## 🔍 ПРОВЕРКА РАБОТЫ

### Backend API:
```bash
# Список грантов
curl http://172.20.10.2:8000/api/v1/grants/

curl http://172.20.10.2:8000/api/v1/grants/1/
```

### Dashboard:
Открой консоль (F12) и создай грант. Должны быть логи:
```
📝 ОТПРАВЛЯЕМ ГРАНТ: {...}
✅ content_uz: "<p>Текст UZ</p>"
✅ content_ru: "<p>Текст RU</p>"  
✅ content_en: "<p>Текст EN</p>"
✅ ГРАНТ СОЗДАН: {...}
```

При редактировании:
```
📰 ЗАГРУЖЕН ПОЛНЫЙ ГРАНТ: {
  id: 1,
  title_uz: "...",
  content_uz: "<p>...</p>",
  title_ru: "...",
  content_ru: "<p>...</p>",
  ...
}
```

### Frontend:
Открой консоль (F12) на странице `/grants`. Должны быть:
```

✅ Гранты загружены: {results: Array(5)}
```
