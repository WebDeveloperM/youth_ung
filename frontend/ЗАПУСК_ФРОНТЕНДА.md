## 🐛 ЕСЛИ НЕ РАБОТАЕТ

### Ошибка CORS:
```
Origin http://localhost:5174 is not allowed by Access-Control-Allow-Origin
```

**Решение:**
```bash
# Перезапусти Django
cd /Users/admin/Desktop/youth_ung/backend
docker compose restart web
```

### Новости не загружаются:
```bash
# Проверь что Backend запущен
curl http://172.20.10.2:8000/api/v1/news/

# Если не работает - перезапусти
cd /Users/admin/Desktop/youth_ung/backend
docker compose up -d
```

### Frontend не запускается:
```bash
# Останови все процессы vite
pkill -f vite

# Запусти снова
cd /Users/admin/Desktop/youth_ung/frontend
npm run dev -- --host --port 5174
```
