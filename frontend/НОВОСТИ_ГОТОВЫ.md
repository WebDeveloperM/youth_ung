## 📊 КАК ЭТО РАБОТАЕТ

### Главная страница:
1. Открывается главная `/`
2. Компонент `NewsListMainpage` загружает последние 3 новости
3. API запрос: `GET /api/v1/news/?limit=3`
4. Показываются новости с изображением, заголовком, анонсом

### Страница всех новостей:
1. Переход на `/news`
2. Компонент `NewsList` загружает все новости
3. API запрос: `GET /api/v1/news/`
4. Показываются все опубликованные новости

### Детальная новость:
1. Клик на новость → переход на `/news/{id}`
2. Компонент `NewsDetail` загружает новость
3. API запрос: `GET /api/v1/news/{id}/`
4. **Backend автоматически увеличивает счетчик просмотров!**
5. Показывается полная новость с комментариями
6. Можно поставить лайк (кнопка ❤️)

### Мультиязычность:
- Компоненты автоматически выбирают язык из `i18n.language`
- Показывают `title_uz`, `title_ru`, или `title_en`
- Аналогично для контента

## 📝 ИСПОЛЬЗОВАНИЕ В КОДЕ

### Пример: Загрузка новостей
```javascript
import { getNewsList } from '@/api/news'

const loadNews = async () => {
  try {
    // Получить последние 3 новости
    const news = await getNewsList({ limit: 3 })
    console.log(news)
    
    // Получить только избранные
    const featured = await getNewsList({ is_featured: true })
    console.log(featured)
  } catch (error) {
    console.error('Ошибка:', error)
  }
}
```

const loadNewsDetail = async (id) => {

## 🌍 МУЛЬТИЯЗЫЧНОСТЬ

Компоненты автоматически используют правильный язык:

```javascript
const { i18n } = useTranslation()

const getTitle = (news) => {
  const lang = i18n.language  // 'uz', 'ru', или 'en'
  return news[`title_${lang}`] || news.title_ru || news.title_en
}
```

Если текст на выбранном языке не найден, используется fallback:
1. Сначала пробует `title_uz` (если язык = 'uz')
2. Если нет, пробует `title_ru`
3. Если нет, использует `title_en`
