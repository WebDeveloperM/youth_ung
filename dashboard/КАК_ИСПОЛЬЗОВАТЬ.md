## 🔑 ВХОД В СИСТЕМУ

1. Открой: `http://172.20.10.2:5173`
2. **НО!** Сначала нужно войти в систему через API

### Получить токен:
```bash
curl -X POST http://172.20.10.2:8000/api/v1/users/sign-in/ \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@admin.com",
    "password": "admin123"
  }'
```

### Или войти через браузер:
1. Открой консоль браузера (F12)
2. Выполни:
```javascript
fetch('http://172.20.10.2:8000/api/v1/users/sign-in/', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'admin@admin.com',
    password: 'admin123'
  })
})
.then(r => r.json())
.then(data => {
  localStorage.setItem('admin_token', data.token);
  console.log('✅ Токен сохранен! Обнови страницу');
  location.reload();
});
```

## 📝 КАК СОЗДАТЬ НОВОСТЬ

### Через Админку (Визуально):

1. **Открой** `http://172.20.10.2:5173`
2. **Войди в систему** (токен должен быть в localStorage)
3. **Перейди** на страницу "Янгиликлар" (News)
4. **Нажми** кнопку "Янги янгилик қўшиш"
5. **Заполни форму:**
   - Загрузи изображение
   - Выбери дату
   - Отметь "Нашр қилиш" (опубликовать)
   - Переключайся между языками (UZ, RU, EN) и заполни:
     - Сарлавҳа (заголовок)
     - Мазмун (контент) - используй Rich Text Editor!
6. **Нажми** "Қўшиш"
7. **Готово!** Новость создана!

### Через API (curl):

```bash
# С файлом
curl -X POST http://172.20.10.2:8000/api/v1/admin/news/ \
  -H "Authorization: Token YOUR_TOKEN_HERE" \
  -F "title_uz=Yangilik" \
  -F "title_ru=Новость" \
  -F "title_en=News" \
  -F "content_uz=<p>Mazmun</p>" \
  -F "content_ru=<p>Контент</p>" \
  -F "content_en=<p>Content</p>" \
  -F "date=2025-11-27" \
  -F "is_published=true" \
  -F "is_featured=false" \
  -F "image=@/path/to/image.jpg"
```

## 🌍 МУЛЬТИЯЗЫЧНОСТЬ

Каждая публикация имеет 3 языка:
- 🇺🇿 **UZ** (Узбекский)
- 🇷🇺 **RU** (Русский)
- 🇬🇧 **EN** (Английский)

### Как заполнить:
1. В форме переключайся между вкладками UZ/RU/EN
2. Заполни заголовок и контент для каждого языка
3. Все поля обязательны!

## 🎯 СТАТУСЫ ПУБЛИКАЦИИ

### Опубликовано (is_published)
- ✅ **Включено** = публикация видна на сайте
- ❌ **Выключено** = черновик, не видно пользователям

### Избранное (is_featured)
- ⭐ **Включено** = показывается на главной
- ⚪ **Выключено** = обычная публикация

## 🐛 TROUBLESHOOTING

### Ошибка: "Failed to load resource"
```bash
# Проверь что Django запущен
docker compose ps

# Перезапусти если нужно
docker compose restart web
```

### Ошибка: "Permission denied"
- Проверь что роль пользователя = **Admin** или **Moderator**
- Проверь что токен в localStorage

### Форма не открывается
- Обнови страницу (Ctrl+R)
- Очисти кэш браузера

### Изображение не загружается
- Проверь размер файла (максимум 10MB)
- Проверь формат (JPG, PNG, WebP, GIF)
