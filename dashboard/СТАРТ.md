## ✏️ РЕДАКТОР ТЕКСТА (Rich Text)

В форме есть полноценный редактор:

### Кнопки редактора:
- **Заголовки** (H1, H2, H3)
- **Жирный** (Bold)
- **Курсив** (Italic)
- **Подчеркнутый** (Underline)
- **Зачеркнутый** (Strike)
- **Списки** (маркированные, нумерованные)
- **Цвета** текста и фона
- **Ссылки** (вставка URL)
- **Изображения** (вставка по URL)
- **Очистить форматирование**

### Как использовать:
1. Выдели текст мышкой
2. Нажми кнопку в панели
3. Применится форматирование!

## 🎯 ВАЖНЫЕ ЗАМЕЧАНИЯ

### ✅ Обязательные поля:
- **Все 3 языка** (UZ, RU, EN) должны быть заполнены
- **Изображение** (обязательно)
- **Дата** (обязательно)

### ✅ Статусы:
- **Нашр қилиш** (is_published):
  - ✅ Включено = видно на сайте
  - ❌ Выключено = черновик
  
- **Танланган** (is_featured):
  - ⭐ Включено = показывается на главной
  - ⚪ Выключено = обычная публикация

### ✅ Изображения:
- Форматы: JPG, PNG, WebP, GIF
- Максимальный размер: 10MB
- Сразу видишь превью

## 🚨 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "Форма не открывается"
**Решение:**
```bash
# Обнови страницу
Ctrl+R

# Проверь консоль (F12) на ошибки
```

### Проблема: "Не могу загрузить изображение"
**Решение:**
- Проверь размер файла (< 10MB)
- Проверь формат (JPG, PNG, WebP, GIF)
- Проверь что Django запущен

### Проблема: "Ошибка при сохранении"
**Решение:**
```bash
# Проверь Django логи
cd /Users/admin/Desktop/youth_ung/backend
docker compose logs --tail=50 web

# Перезапусти Django
docker compose restart web
```

### Проблема: "Не могу войти"
**Решение:**
```bash
# Создай нового админа
docker compose exec web python manage.py createsuperuser \
  --email admin@admin.com \
  --username admin

# Пароль: admin123
```
