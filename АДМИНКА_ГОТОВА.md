## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Перезапустите Django сервер:
```bash
docker-compose restart web
```

### 2. Проверьте админские API:
```bash
# Технологии
curl -H "Authorization: Token YOUR_TOKEN" http://localhost:8000/api/v1/admin/technologies/

# Проекты
curl -H "Authorization: Token YOUR_TOKEN" http://localhost:8000/api/v1/admin/projects/

# Исследования
curl -H "Authorization: Token YOUR_TOKEN" http://localhost:8000/api/v1/admin/research/

# Результаты
curl -H "Authorization: Token YOUR_TOKEN" http://localhost:8000/api/v1/admin/results/
```

### 3. Теперь в админ-панели (Dashboard) можно:

**Создать страницы** (похожие на News, Grants и т.д.):
- `Technologies.tsx` - управление технологиями
- `Projects.tsx` - управление проектами  
- `Research.tsx` - управление исследованиями
- `Results.tsx` - управление результатами

**Структура страницы** (пример для Technologies.tsx):

```typescript
import { useState } from 'react';
import ContentListPage from '../components/common/ContentListPage';
import { contentAPI } from '../api';

const Technologies = () => {
  const columns = [
    { key: 'id', label: 'ID' },
    { key: 'title_ru', label: 'Заголовок' },
    { key: 'category', label: 'Категория' },
    { key: 'date', label: 'Дата' },
    { key: 'is_published', label: 'Опубликовано' },
  ];

  return (
    <ContentListPage
      title="Технологии"
      description="Управление технологиями"
      api={contentAPI.technologies}
      columns={columns}
      searchPlaceholder="Поиск технологии..."
      onAddNew={() => {/* открыть форму */}}
    />
  );
};

export default Technologies;
```
